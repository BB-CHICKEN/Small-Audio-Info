MoeKoe Music 音乐信息显示插件（增强版）
简介
这是一个为 MoeKoe Music 桌面版（基于 Electron）开发的扩展插件。它在播放器界面的右下角实时显示当前歌曲的音频采样率（如 44100 Hz）、位深度（如 16bit）以及文件大小（自动转换为 B、KB、MB、GB），让您快速了解当前播放的音乐文件质量与体积。

与旧版依赖本地 API 不同，本插件通过直接解析音频文件头部（仅读取前 16KB 数据）获取信息，无需额外网络请求，兼容本地文件与网络流，响应迅速且准确。

功能特点
🎵 智能解析：自动识别 FLAC 和 MP3 格式，提取采样率与位深度（FLAC 支持位深度解析，MP3 仅提供采样率）。

📡 无 API 依赖：不再依赖 MoeKoe Music 的内部 HTTP API，直接从歌曲的 url 发起 Range 请求，轻量高效。

📍 位置固定：显示在播放器容器（.player-container）的右下角，不干扰主要控制按钮。

📦 实时更新：每 1 秒自动检测当前歌曲变化，切歌后立即刷新信息。

🛡️ 容错性强：若解析失败或歌曲无法访问，会显示“未知”或“加载中…”，不影响播放器正常运行。

安装方法
下载插件压缩包（或直接获取源码）
（若需发布，请参阅 Releases）

打开 MoeKoe Music，进入 设置 → 插件管理，点击“安装插件”，选择下载的 .zip 文件或文件夹。

安装后重启 MoeKoe Music，插件即自动生效。

提示：插件无需任何配置，启动后自动在播放器右下角显示信息。

使用说明
播放任意歌曲（本地文件或网络流），右下角会出现信息卡片。

格式示例：

44100 Hz · 16bit（FLAC 文件）

44100 Hz（MP3 文件）

文件大小如 12.34 MB

若歌曲格式非 FLAC/MP3，或数据不完整，则显示“未知”。

注意事项
插件依赖浏览器 fetch API 发起 Range 请求，确保 MoeKoe Music 运行在允许跨域的环境（通常 Electron 默认允许）。

仅当歌曲的 url 可访问且服务器支持 Range 请求时，才能正确获取文件大小与头部信息。

对于某些流媒体或受限 URL，可能无法获取信息，此时会显示“未知”。

技术细节（供开发者参考）
解析逻辑：

FLAC：检查 fLaC 标记，解析 STREAMINFO 块获取采样率和位深度。

MP3：检查 ID3v2 标签偏移，随后查找同步帧（0xFFE0）提取采样率索引。

更新机制：通过 setInterval 每 1 秒调用 updateDisplay()，比较当前歌曲 hash/url 是否变化，避免重复请求。
